1. body내용 가져오기

html() : body의 모든 내용 가져오기 : 태그를 포함한다.

text() : body의 모든 내용 가져오기 : 태그를 포함하지 않는다.




2. box2내용 가져오기

html() : box2의 모든 내용 가져오기 : 태그를 포함한다.

text() : box2의 모든 내용 가져오기 : 태그를 포함하지 않는다.




3. box3의 p태그의 내용 가져오기

html() : p의 모든 내용 가져오기 : 태그를 포함한다.
-> p 태그가 여러개일 경우에는 첫번째 요소만 가져온다.
-> 반복문을 사용해서 모두를 다 가져온다.
: $.each(요소,function(v1, v2){ })
: $(요소).each(function(v1, v2){ })
text() : p의 모든 내용 가져오기 : 태그를 포함하지 않는다. 여러개인경우 모든 p태그의 문자만
